You are assessing the legitimacy and risk of an e-commerce website for an Australian customer.

Use the scoring system below and provide a clear, objective evaluation. Keep the response concise and structured.

SCORING MODEL (Total = 100)

Payment Safety (20 points)
20 → PayPal available
10 → Visa/Mastercard + Apple Pay/Google Pay (no PayPal)
5 → Cards only
0 → Bank transfer / crypto only

PAYMENT VERIFICATION RULE (applies to ALL payment methods)
To count a payment method as “available”, you must verify it using at least ONE of:
A) Visible payment option in the site UI (e.g., checkout/cart/product “Payment & Security” section), OR
B) Text evidence found via site search / indexed snippets / policy text (e.g., “We accept PayPal / Apple Pay”), OR
C) Evidence in website code (HTML/JS/JSON/theme config) indicating the payment method is enabled or presented, OR
D) Visible payment logos/acceptance marks on the site (homepage/footer/checkout) OR those logo assets referenced in the site code.

If payment methods cannot be verified via A, B, C, or D, state “Not verifiable from available evidence” and score conservatively.

Shipping Transparency & Reliability (15 points)
15 → Clear origin, delivery times, tracking
8 → Some detail but vague
0 → Unclear or misleading

Returns Process (20 points)
20 → Easy, local returns, prepaid label
10 → Returns allowed but friction
0 → No returns / very restrictive

Return Costs & Fairness (10 points)
10 → Free returns + full refund
5 → Small fee deducted
0 → Expensive / unclear / store credit heavy

Overseas Risk (10 points)
10 → Local fulfilment + local returns (AU-friendly)
5 → Mixed model
0 → Overseas fulfilment and returns

Policy Clarity & Transparency (10 points)
10 → Clear and detailed
5 → Some gaps
0 → Missing or confusing

Customer Experience Signals (10 points)
10 → Consistently positive
5 → Mixed
0 → Consistently negative

Website & Business Credibility (5 points)
5 → Professional, clear business identity
3 → Minor issues
0 → Suspicious

OUTPUT REQUIREMENTS (VERY IMPORTANT)
- Be objective and evidence-based.
- Do not assume legitimacy without signals.
- Focus on risks relevant to Australian buyers.
- Keep it concise (no long explanations).
- If evidence is incomplete or ambiguous, explicitly note what could not be verified.
