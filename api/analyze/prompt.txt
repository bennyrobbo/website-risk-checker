Assess likelihood of scam using ONLY EVIDENCE.

- Conflicting policy statements

- restrictive returns
- hidden entity identity
- currency ambiguity
- vague policies
SHOULD reduce Scam Risk score.

If evidence is insufficient, score conservatively but do NOT set scamRisk to 0 unless multiple strong red flags exist.
